Project Name: Digitools-Platform

Description: DigiTools is a modern React-based web application where users can explore and purchase premium digital tools designed to boost productivity and creativity. It features a clean UI, dynamic cart system, and smooth user experience.

Technologies Used:
React.js
Tailwind CSS
DaisyUI
JavaScript (ES6+)
React Toastify
React Icons

Features:
🛒 Dynamic Cart System – Add and remove products with real-time updates
🔄 Tab Switching UI – Easily switch between Products and Cart view
⚡ Fast & Responsive Design – Optimized for all devices with smooth performance
